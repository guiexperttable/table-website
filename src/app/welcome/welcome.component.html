GuiExpert Table

The agnostic table component for building the future

Fully-featured and highly customizable data grid.

It delivers outstanding performance, has no third-party dependencies.

Available as component for Angular, Vue.js, React, Preact, Svelte, Solid, Web Component and available as plain JavaScript function.

Features

  • Excellent performance for large tables by vertical and horizontal virtual scrolling
  • Column Interactions (resize, reorder)
  • Sorting Rows
  • Row, Column and Range Selection
  • Single and Multi Selection

Interaction

  • Tree table (Hierarchical View)
  • Accessibility support: Keyboard Shortcuts
  • Custom Filtering
  • In-place Cell Editing
  • Userdefined Key and Mouse Events

Layout, Look & Feel

  • Customizable Cell Contents via Renderer
  • Customizable Look & Feel (via css variables)
  • Row sorting)
  • Column Reordering (Drag and Drop)
  • State Persistence (Row Sorting, Column Order, Selection)

Usability

  • Row and Column Spanning
  • Fixed Columns (Left and Right)
