GuiExpert Table - API



TableModels

The table model is the most important part. It defines the structure of the table: row count, column count, value in a cell, etc.

TableModelFactory

Helper class for creating table models out of data structures.
  • This factory class has some static methods, which help in creating table models.
  • chevron_right buildByTypedRowsParam An array of business objects can easily be converted into a table model by this method.
  • chevron_right createByArrayOfArrayParams An array of arrays data structure can be easily converted into a TableModel.
  • chevron_right createByAreaModelsParam If area models (header/body/footer) are built, they can be converted into a table model.
  • chevron_right buildTreeRows An object array (T[]) can be grouped into a tree structure by this method. All objects will be replaced by a TreeRow<T>
